  • Div tag The div tag or division tag is used to make blocks or divisions in the document. 
    <div> This is div block </div>;
  • Span tag The span is a container for inline content 
    <span> This is span block </div>;
  • p tag The p tag is used to create a paragraph in HTML 
    <p> This is a paragraph <p>;
  • pre tag The pre tag represents pre-formatted text 
    <pre> This is span block </pre>;
  • code tag The code tag is used to represent source codes in HTML 
    <code> This is span block </code>;

    Text Formatting

    Text formatting tags are used to format text or data in HTML documents. You can do certain things like creating italic, buld, and strong text to make your document look more attractive and understandable.
  • <b> tag: 
    <b> I am buld tag </b>;
  • <strong> tag: 
    <strong> I'm important text </strong>;
  • <i> tag: 
    <i> I'm italic text </i>;
  • <em> tag: 
    <em> Emphasized text </em>;
  • <sub> tag: 
    <sub> Subscript </sub>;
  • <sup> tag: 
    <sup> Superscript </sup>;

    Lists

    Lists can be either numerical, alphabetic, bullet, or other symbols. You can specify list type and list items in HTML for a clean document.
  • <ol> tag: 
    <ol> 
        <li>Data 1</li>
        <li>Data 2</li>
        <li>Data 3</li>
 </ol>;
  • <ul> tag: 
    <ul> 
        <li>Data 1</li>
        <li>Data 2</li>
        <li>Data 3</li>
 </ul>;
  • Table A table is a collection of rows and columns. It is used to represent data in tabular form.

                <th colspan="2">Column2</th>
            </tr>

            </tr>
